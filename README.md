GuideMe — видеть мир ушами
Версия: alpha 1.8 (расширенные голосовые команды)
Дата: 04.04.2026

ОПИСАНИЕ
--------
GuideMe — голосовой помощник для незрячих и слабовидящих.
Приложение через камеру телефона распознаёт объекты, людей, машины,
определяет расстояние до них и направление, а затем голосом сообщает
пользователю обстановку вокруг. Управление — голосом (ключевое слово "Лина").

ОСНОВНЫЕ ВОЗМОЖНОСТИ alpha 1.8
------------------------------
- Распознавание 20+ типов объектов (люди, машины, животные, предметы)
- Определение расстояния до объекта (после калибровки)
- Определение направления: слева, справа, прямо
- Группировка до 3 объектов в одной фразе
- Предупреждения об опасных объектах (очень близко) + вибрация
- Голосовые подсказки с ускорением 20% (быстрая речь)
- Автоматический запрос разрешений при запуске (камера + микрофон)
- Мгновенная остановка речи при выключении камеры

ГОЛОСОВЫЕ КОМАНДЫ (после слова "Лина")
---------------------------------------
Основные команды:
  • "что видишь" / "описание" — голосовое описание текущей сцены
  • "калибровка" — калибровка расстояния (встаньте на 1 метр от камеры)
  • "стоп" / "останови" — выключить камеру
  • "старт" / "запусти" — включить камеру

Настройки:
  • "громкость 50" — установить громкость в процентах (0–100)
  • "громче" / "тише" — увеличить/уменьшить громкость на 10%
  • "частота 3" — интервал голосовых подсказок (3–10 секунд)
  • "уверенность 0.5" — порог уверенности распознавания (0.3–0.9)
  • "опасность 2 метра" — порог опасности в метрах (0.5–3)
  • "тихий режим" / "звук выключить" — отключить звук
  • "звук включить" — включить звук
  • "вибрация выключить" / "вибрация включить" — управление вибрацией

Дополнительные команды:
  • "статус" / "что настроено" — озвучить текущие настройки
  • "замолчи" / "стоп аудио" — остановить текущую речь
  • "помощь" — список всех команд

ЗАПУСК ПРИЛОЖЕНИЯ
-----------------
1. Скопируйте все файлы проекта на веб-сервер или локальный диск.
2. Откройте файл index.html в современном браузере (Chrome, Edge, Safari).
3. Разрешите доступ к камере и микрофону при запросе.
4. Дождитесь загрузки модели распознавания (статус "Модель: готова").
5. Скажите "Лина, старт" или нажмите кнопку "Запустить камеру".

ИСПОЛЬЗОВАНИЕ
-------------
- После запуска камеры приложение автоматически голосом называет объекты
  (например, "человек слева близко").
- При приближении опасного объекта — предупреждение и вибрация.
- Если несколько объектов — группирует: "машина справа, человек прямо и собака".
- Калибровка: встаньте на расстоянии 1 метра от камеры так, чтобы вас было видно.
  Скажите "Лина, калибровка". Приложение определит человека и сохранит настройку.
- Все настройки сохраняются в браузере.

НАСТРОЙКИ (кнопка ⚙️ или команда "Лина, настройки")
---------------------------------------------------
- Громкость: ползунок 0–1 (или голосом "громкость 70")
- Частота подсказок: 3–10 секунд
- Минимальная уверенность: 0.3–0.9
- Порог опасности: 0.5–3 метра
- Звук вкл/выкл
- Вибрация вкл/выкл

СТРУКТУРА ФАЙЛОВ
----------------

css/style.css       — стили (адаптированы для незрячих)
audio/              — папка с mp3-файлами голосовых фраз

  vision/           — распознавание: camera.js, detection.js, object-tracker.js
  features/         — функции: calibration.js, safety.js, navigation.js
  ui/               — интерфейс: ui-core.js, ui-controls.js, ui-settings.js
  utils/            — утилиты: helpers.js
  main.js           — главная логика приложения

ТРЕБОВАНИЯ
----------
- Браузер с поддержкой WebRTC (камера), Web Audio API и SpeechRecognition.
- Для голосовых подсказок необходима папка audio/ с mp3-файлами.
- Рекомендуется использовать устройство с камерой (смартфон, ноутбук).

ИЗВЕСТНЫЕ ОСОБЕННОСТИ
---------------------
- Для точного определения расстояния требуется калибровка.
- Распознавание работает лучше при хорошем освещении.
- Голосовые команды требуют разрешения микрофона.
- В некоторых браузерах (Safari) может потребоваться разрешение на автозапуск аудио.
